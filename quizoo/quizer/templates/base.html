WHAT Are We?

Quizoo is a platform created with close reference to the current academic situation of schools and colleges due to the Covid-19 pandemic. We have have found that the currently available platforms are not yet ready for conducting examinations as they crash or run into errors minutes before it starts.

With Quizoo, creating tests becomes easy. It is a great tool and extremely user friendly. We provide multiple features like:

  • Auto email students their scores after the test gets over
  • Show questions in random order and shuffle options
  • Show students their scores during the test itself
  • Reattempt a test if missed by any chance and admin permits
  • Allow admin to retake a previous quiz
  • Multiple admins for a quiz
  • Camera mode On or Off
  • Generating a Random Quiz from Previously owned question set
  • Give x amount of questions to each user from a pool of n questions
